Java基础（四）–数组与方法

一、数组

1.1 数组的基础定义

数组是内存中的一段空间，可以理解为一个“盒子”，里面按照顺序存放很多类型相同的数据，比如学生的成绩、科目信息等等。数组中的元素都可以通过下标来进行访问，需要注意的是，下标是从0开始的，对应的是数组中第一个元素。

1.2 如何使用Java中的数组
  • 声明数组

语法：数据类型[] 数组名 或者 数据类型 数组名[]

（数组名可以使任意合法的变量名）

  • 分配空间

指定数组中最多可以存储多少个元素

语法：数组名 = new 数据类型[数组长度]

不过也可以直接将声明和分配进行合并，同时进行

  • 赋值

创建成功之后就可以存放数组了

1.1 如何定义方法

语法：访问修饰符 返回值类型 方法名(参数列表){方法体}

主要是包含下面四种：

  • 无参无返回值
  • 无参带返回值
  • 有参无返回值
  • 有参带返回值
